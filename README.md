📊 Análisis Exploratorio de Datos (EDA)
📌 Descripción

Este repositorio contiene el desarrollo de análisis exploratorio de datos como parte de mi proceso de formación en un Bootcamp de Análisis de Datos.

El objetivo principal es aplicar técnicas de exploración, limpieza y comprensión de datos para identificar patrones, detectar anomalías y generar insights relevantes que sirvan como base para análisis más avanzados.

🎯 Objetivos
Comprender la estructura y calidad de los datos
Realizar procesos de limpieza y transformación
Identificar patrones, tendencias y outliers
Generar visualizaciones que faciliten la interpretación de los datos
🛠️ Tecnologías utilizadas
Python
Google Colab
Librerías principales:
Pandas
NumPy
Matplotlib
Seaborn
📂 Estructura del repositorio
├── notebooks/        # Análisis exploratorio en notebooks
├── data/             # Conjuntos de datos utilizados
├── outputs/          # Gráficos y resultados generados
└── README.md         # Documentación del proyecto
▶️ Uso

Puedes visualizar los notebooks directamente en Google Colab o clonar el repositorio:

git clone https://github.com/tu-usuario/tu-repositorio.git

Luego abre los archivos .ipynb en Google Colab para ejecutar el análisis paso a paso.

📊 Contenido
Carga y exploración inicial de datos
Limpieza y tratamiento de valores nulos
Análisis estadístico descriptivo
Visualización de variables
Identificación de patrones relevantes
